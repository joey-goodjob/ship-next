# BeatMV Section Topology

Only one source section is in scope.

1. Creators gallery
   - Source section text: "See What Creators Are Making"
   - Flow position on BeatMV: second homepage section after the hero.
   - Local destination: homepage after the existing Hero block and before Features.
   - Structure: dark full-width section, constrained inner header, two clipped marquee rows.
   - Interaction model: time-driven marquee.
   - Dependencies: local MP4 assets in `public/beatmv-showcase/`, local background image `bgdark.webp`, CSS keyframes in `src/app/globals.css`.
