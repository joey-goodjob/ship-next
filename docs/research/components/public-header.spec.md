# Public Header Specification

## Overview
- Target files: `src/components/site-header.tsx`, `src/blocks/header.tsx`
- Screenshot: `docs/design-references/lyricedits.ai/home-desktop.png`
- Interaction model: static nav, mobile menu in existing component.

## Styles
- Height: 72px.
- Background: white.
- Logo: black wordmark with yellow suffix/accent.
- Desktop nav: right-aligned, 14px, slate muted text; active pricing yellow.
- Primary button: yellow `rgb(251, 191, 36)`, black text, 10px radius.
- Login button: light gray fill.

## Responsive
- Desktop: logo left, nav/actions right.
- Mobile: keep compact menu but preserve logo and yellow CTA styling.
