### 全站原则


### · 转化五原则：①工具嵌入页面不做介绍页 ②注册墙在"生成后、下载前" ③首屏动态demo ④声明配数字/功能跟benefit ⑤CTA动词开头+否定句消摩擦（No signup. No watermark. No credit card.）


### · GEO三件套：每页开头直接回答式段落 / 对比表或参数表 / FAQ答案第一句给结论


### · 文案风格8条：短句(>20词拆) / CTA动词开头 / 声明配数字 / 功能跟benefit / 疑虑就地解决 / 否定句消摩擦 / 竞品页必有"对手更好"段 / CTA措辞渐进(首屏祈使→中部场景→底部情绪)


### · 埋点最小集：upload_start → upload_complete → lyrics_reviewed → style_selected → video_generated → signup_complete → download。基准漏斗 100/30/20/12/10


### · Schema速查：首页SoftwareApplication / 工具页+FAQPage / 教程页HowTo+FAQPage / 博客Article+FAQPage / 全站BreadcrumbList


### · 性能红线：LCP<2.0s / 首屏<200KB / 模板预览用WebM非GIF且lazy load / 所有媒体预设宽高比
